# Naming convention

---

### File / Folder
- *.java* named after the class they hold and in **UpperCamelCase**
- *Asset* file (png, svg, mp3, gif, ...) in **snake_case**
- *Folder* in **lower camelCase**

---

### Variable (lower camelCase)

---

### Constant (UPPER_CASE)
#### Private Constant (_UPPER_CASE)
Constant in java = **static final**

---

### Function (lower camelCase)

---

### Namespace (UpperCamelCase)

---

### Class / Struct
- *name* in **UpperCamelCase** | ✅ `EntitySystem`
- *public propritie* in **lower camelCase** | ✅ `numberOfEntity`
- *protected propritie* in **p_camelCase** | ✅ `p_canJump`
- *private propitie* in **_camelCase** | ✅ `_canJump`
- *(private, protected, public) methods* in **lower camelCase** | ✅ `useAction(...)`

---

### Interface
Always named "I{insert name **UpperCamelCase**}" | ✅ `IComponent`

---

### Abstract Class
Always named "Abstract{insert name **UpperCamelCase**}" | ✅ `AbstractComponent`
